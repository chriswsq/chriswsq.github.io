docker-compose 安装7.5.4 grafana 连接zabbix

2021-04-23

先docker run将grafana的配置文件拷贝出来

docker run -d --name grafana -p 3000:3000 grafana/grafana:7.5.4
docker cp  grafana:/etc/grafana/grafana.ini  ./grafana
docker cp grafana:/var/lib/grafana/ ./grafana/

然后删除该容器
docker rm -f grafana

version: '3.3'
services:
  grafana:
    image: grafana/grafana:7.5.4
    container_name: grafana
    restart: always
    privileged: true
    user: root
    ports:
      - 3000:3000
    environment:
      - GF_INSTALL_PLUGINS=grafana-clock-panel,grafana-simple-json-datasource
    volumes:
      - ./grafana/grafana.ini:/etc/grafana/grafana.ini
      - ./grafana:/var/lib/grafana
      - /etc/localtime:/etc/localtime
      #log目录，后期会写入log文件
      - ./log:/var/log/grafana

运行服务
docker-compose up -d

根据需求到这里去安装插件
https://grafana.com/grafana/plugins/

例如zabbix

点击如下图标，进入zabbix插件页面

执行安装zabbix插件命令 grafana-cli plugins install alexanderzobnin-zabbix-app
安装完毕后重启下grafana

需要登录grafana页面进行开启插件

在zabbix添加用于grafana的只读用户名
添加用户组 名称grafana

添加用户，归到grafana组中

权限类型为用户

下面在grafana上编辑zabbix数据源，填写相关信息，保存

因为这里使用docker部署的zabbix和grafana所以url为
http://{{ IP }}:8088/api_jsonrpc.php
